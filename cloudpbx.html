National Cluster Monitoring & Alerting DevOps @ CloudPBX Inc. (2017)

CloudPBX Inc. is a cloud based voice over internet protocol (VoIP) phone company, where phone calls connections are established over the internet, with data being set using IPs instead of telephone lines.

Main tasks include:
i. Installed and tested monitoring and statistics collection tools and plugins on DigitalOcean Ubuntu via CLI
ii. Developed bash scripts to perform traceroute and mtr tests, importing CSV data into PostgreSQL
iii. Analyzed performance data collected from key system metrics to create graphs and generate insights to support key business decisions
iv. Created maps to showcase distance between clients and datacenters to improve operational efficiency based on location of IP addresses using SQL queries
v. Separated probe and database logic to prepare for production on various probes (BusyBox, EdgeOS)
