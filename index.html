BattleScripts



Battlescripts is a group project created using HTML, CSS, JS, Laravel PHP framework, Unity, MySQL database, hosted via Google Cloud Platform.

The game introduces coding concepts to players and sharpen their ability to build programs using blocks of logic without having to worry about syntax or typing errors. Players will have to think strategically through the use of various different cards while keeping an eye on their own and their opponent's life points. Players will have the choice of joining a random room, or creating and joining an existing room online.

Each player will have five cards drawn. The cards are blocks of code which will be added to that player's program. Players take turns drawing and playing cards from a deck, with the choice of executing, storing, or building on each turn. They may strengthen themself, powerup attacks on opponents, or have the option of storing useful blocks of code.

FOO and BAR bars are the life points, and energy points of each player. Everytime the player executes a program, it will consume a certain amount of BAR points for each piece of code added. BAR will be replenished at the start of each turn. Each player starts with zero bugs and when one player overflows the other player's points (either over or under), then their bugs amount will increase by one.

HandyMan



HandyMan is a mobile app prototype designed to be a platform for people to lookup guides and tutorials on home repair using Axure RP. Users will have to login or create an account before accessing other parts of the application. Once logged in, users will have the option of viewing tutorials (in the form of text, words, or augmented reality), or access the discussion forum to post and answer questions. If these are not succificient, then we have provided users the possibility of contacting an expert directly.

Mount Monstrosity



Mount Monstrosity is a simple shooting game created using Processing. The game has three levels, each increasing in difficulty. The ultimate goal is to defeat the boss that eventually appears in level three. The player can move the character and shoot out bullets using arrow keys and space-bar.
