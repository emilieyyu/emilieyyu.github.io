CloudPBX

My first Co-op Work Experience was completed at CloudPBX Inc., a cloud based voice over internet protocol (VoIP) phone company, where phone calls connections are established over the internet, with data being set using IPs instead of telephone lines. It was a memorable experience, giving me the opportunity to really explore in areas that I was familiar with (research) and learn new things in various aspects (command line, virtual servers, database, networking). What I have been able to accomplish in 4 months can be broken down into 4 categories:

i) research based on given list of software that provide monitoring and alerting features

ii) install chosen combinations of software and plugins on test servers

iii) based on the top selected combination, write scripts and import data into database to produce graphs and maps showing key performance indicators (KPIs)

iv) separate probe and database logic in preparation for production using numerous probes

SFU BookClub

SFU BookClub is a group project created using HTML, CSS, JavaScript, and Ruby on Rails. The main purpose of our web application project is to create a platform for SFU students to buy and sell textbooks or other resources with their peers. Students will be able to create an account and post textbooks for sale. They will also be able to comment on other students post to ask for details or questions of their post.

Mount Monstrosity

Mount Monstrosity is a simple shooting game created using Processing. The game has three levels, each increasing in difficulty. The ultimate goal is to defeat the boss that eventually appears in level three. The player can move the character and shoot out bullets using arrow keys and space-bar.

  • Video Link: Mount Monstrosity

Whack-a-Mole

This Whack-a-Monster game was created using object orient programming, Processing. The purpose of this game is for users to interact (hit) the monsters moving on the screen with their mouse. As the monsters are hit, they will disappear through animations and the score of the player will increase, which can be seen from the top left hand corner. Use the mouse to destroy the monsters by clicking and aiming with base of the sword.

  • Website: OpenProcessing

Interact with Spongebob

This project was created using Processing. Interact with Spongebob on the screen by moving the mouse from left to right and from top to bottom, or vice versa. Witness how different Spongebob responds differently to these actions. Possible results are: shrinking or increasing in size, rotations of the body, as they move from one side of the screen to the other.

  • Video Link: OpenProcessing
